Tydzień 3

W tym tygodniu:

  • metodologia Kanban (zawsze wiedziałem że ten cały multitasking jest przereklamowany)
  • HTML, a tam między innymi tworzenie linków i łączy wewnętrznych, wstawianie obrazków, tworzenie nawigacji
  • konwencje nazewnictwa:
    • camelCase, np mySuperBoldHeader używany m. in. w JavaScript
    • kebab-case, np my-super-bold-header np: nazywanie plików w katalogach, atrybuty id oraz class
    • snake_case np my_super_bold_header
    • PascalCase np MySuperBoldHeader
  • zasady nadawania nazw:
    • nazwy bez polskich znaków
    • nie używamy spacji
    • najlepiej używać angielskich słów
    • trzymamy się konsekwentnie jednego schematu
